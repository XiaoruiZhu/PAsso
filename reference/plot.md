# A matrix of partial regression plots between responses after adjustments

A plot matrix to display the results of partial association analyses.
Upper-triangle contains scatter-plot matrix between each pair of
response variables. Lower-triangle contains the partial correlation
coefficients adjusted by covariates.

## Usage

``` r
# S3 method for class 'PAsso'
plot(x, color = "#444444", shape = 19, size = 2, alpha = 0.5, ...)
```

## Arguments

- x:

  The object in "PAsso" class that is generated by "PAsso" or "test".

- color:

  The color of points.

- shape:

  The shape of points. For more details see the help vignette:
  [`vignette("ggplot2-specs", package = "ggplot2")`](https://ggplot2.tidyverse.org/articles/ggplot2-specs.html)

- size:

  The size of points. For more details see the help vignette:
  [`vignette("ggplot2-specs", package = "ggplot2")`](https://ggplot2.tidyverse.org/articles/ggplot2-specs.html)

- alpha:

  The value to make the points transparent. For more details see the
  help vignette:
  [`vignette("ggplot2-specs", package = "ggplot2")`](https://ggplot2.tidyverse.org/articles/ggplot2-specs.html)

- ...:

  Additional optional arguments to be passed onto.

## Value

A `"GGally"` object.

## Details

A pairwise plot matrix reveals the partial association between ordinal
variables. All the plots are based on surrogate residuals generated from
`"resides"` function. Graphics are designed based on
[`ggplot2`](https://ggplot2.tidyverse.org/reference/ggplot2-package.html)
and `"GGally"`.
